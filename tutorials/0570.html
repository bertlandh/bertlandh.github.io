Installing Batocera on S912 Android TV Box (Tanix TX9s)

Things You Will Need

  • A microSD card at least 4 GB https://amzn.to/36CpShC | https://amzn.to/3CXJngt | https://amzn.to/3JqyxSP

Downloads and Flashing to MicroSD

  1. Download Batocera Download
  2. Download Balena Etcher Download
  3. Run Balena Etcher
  4. Burn Batocera image to microSD card

  8. meson-gxm-q201.dtb = s912
    meson-gxm-q200.dtb = s912

  9. Navigate back to the root of the SD card and into /extlinux
  10. Edit the extlinux.conf file in a text editor
  11. Paste the full dtb file name on the FDT line
    FDT /boot/meson-gxm-q201.dtb
  12. Safely remove the SD card from the PC

Booting to Batocera for the First Time

  1. Insert the SD card into the Android TV box
  2. With an unfolded paperclip, press and hold the reset button (sometimes hidden inside the 3.5mm AV port)
  3. While still holding the reset button, plug the power adapter into the Android TV box
  4. You should see the Batocera splash screen
  5. Setup Your Controller
  6. Game
