Sideload Aptoide App Store Onto Ouya Console

  1. On the Ouya, go to Manage > System > Advanced
  2. Scroll down and select Security
  3. Scroll down and check the Unknown sources box > Select OK to confirm
  4. Return to the home screen and select Make > Software > Browser
  5. Navigate to https://aptoide.com/
  6. Wait for just a moment and the download link will show at the top of the page
  7. Hover toward the middle right of the area to find the download button (in the Ouya browser it doesn't display)
  8. Click the invisible Download button and the download should start
  9. Select Browser to complete the download action
  10. Click back a few times to return to the Make > Software area
  11. Select an installed file browser application such as ES File Explorer
  12. Navigate to Download to find the downloaded Aptoide APK
  13. Select the .apk file and select Install
  14. Alternatively, if FilePwn is installed on the Ouya return to the home screen and select Play > FilePwn
  15. Navigate to Download to find the downloaded Aptoide APK
  16. Select the .apk file
  17. Select Package installer to complete the install action
  18. Click Install and wait for the installation to complete
