Install Highlighted Cursor Theme for Gnome

  1. Download the bDMZ Cursor theme Download
  2. Extract the downloaded archive with the following command:
    cd ~/Downloads
    mkdir bDMZ
    # extract the downloaded tar.gz
    sudo tar xzvf *bDMZ*tar.gz -C ~/Downloads/bDMZ --strip-components=1
    # copy the extracted cursor themes to /usr/share/icons
    sudo mv ~/Downloads/bDMZ/* /usr/share/icons
    # install gnome tweaks utility
    sudo apt install gnome-tweaks
  3. Launch Gnome Tweaks
  4. Select Appearance from the left menu
  5. Click the dropdown across from Cursor and select the new cursor theme to use
